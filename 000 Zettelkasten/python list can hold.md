# python list can hold

[[Lists]] in [[Python]] not only hold objects of same type, but can also hold any type

```py

things = [
	maths,  # a module
	math.factorial # a function
	int, # a type
	nums.append # a menthod
	nums, # a list
	KeyError("Error!") # an exception
	KeyError, # an exception class
]
```
