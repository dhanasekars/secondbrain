# Objects
```
Primitives that represents data
```

You can find the object type by 

```
>a = 5
>type(a)
```

Change type by *typecasting* 

```
>float(a)
```

### Scalar objects :
Objects that cannot be subdivided
example : int, float, bool , NoneType
